# MANUAL IRRIGATION BENCH CARD

## LED Meaning

LED ON

* Valve open
* Irrigation active

LED OFF

* Valve closed
* Irrigation inactive

## Manual Irrigation

Press and hold button:

* Valve opens

Release button:

* Valve closes

## Emergency Stop

Serial command:

e

Result:

* Valve closes immediately

## Status Command

Serial command:

s

Shows:

* Valve state
* Uptime
* Last irrigation event
