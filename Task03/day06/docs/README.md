# Phase 1 Demo Steps

1. Power on controller.
2. Verify LED is OFF.
3. Verify valve is closed.
4. Hold manual button.
5. Confirm relay click and valve open.
6. Release button.
7. Confirm valve closes.
8. Send serial command 's'.
9. Verify status output.
10. Send serial command 'e'.
11. Verify emergency stop.
